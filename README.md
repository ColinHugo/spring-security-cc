# Spring Security

## Urls disponibles

### Iniciar sesión
```http://localhost:8080/auth/authenticate```

### Obtener todos los productos - Se necesita token
```http://localhost:8080/products```

### Registrar product productos - Se necesita token
```http://localhost:8080/products```
